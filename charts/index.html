ImageEngine Helm Charts

Cloud-native image optimization, packaged for Kubernetes.

Add the repository

helm repo add imageengine https://kube.imageengine.io/charts
helm repo update
helm search repo imageengine

Install

helm install my-imageengine imageengine/imageengine-kube

Verify signatures

Charts published from version 0.8.5 onward are signed with GPG.

curl -fsSL https://kube.imageengine.io/charts/pubkey.asc \
  | gpg --dearmor > ~/.gnupg/imageengine-kube.kbx

helm install my-imageengine imageengine/imageengine-kube \
  --verify --keyring ~/.gnupg/imageengine-kube.kbx
